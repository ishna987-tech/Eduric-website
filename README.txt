Eduric Static Website
====================

Quick start (Windows):
1) Unzip into a folder (e.g., C:\eduric-site)
2) Open index.html in a browser

Optional: run a local server (recommended for correct relative paths):
- If you have Python installed:
  python -m http.server 8080
  Then open: http://localhost:8080

Deploy:
- Upload the entire folder contents to any static host (Cloudflare Pages, Netlify, GitHub Pages, S3, etc.)
- Set your domain to point to the host.
